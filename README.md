# Python Tic Tac Toi

python Tic Tac Toi with tkinter small project

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install PyQt5.

```bash
pip install PyQt5
```

## Usage

```python
import PyQt5

uic.loadUi("tictactoi.ui",self) //check the path of ui file
```
